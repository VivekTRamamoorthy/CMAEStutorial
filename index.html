How CMA-ES works?

  1. CMA-ES is a population-based heuristic strategy which improves the current set of solutions through an evolutionary process to reach the optimal solution
  2. It uses a multi-variate Gaussian to represent the population in a generation.
  3. In this case the initial guess is set to be \( \mathbf{x}^{mean} = [1, 1]^T \) with a standard deviation of \( 0.3 \) for both \( x_1\) and \(x_2\).
  4. In each generation,
    • CMA-ES generates sample points \( [x_1,x_2]^T \)
    • Updates the covariance matrix using the \(f \) values obtained at the sampled points
    • Updates the mean value of the sampling distribution toward the direction of improvement
  5. The process is repeated until a satisfactory result is achieved.
